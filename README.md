# Website Portfolio Sederhana

## Deskripsi
Proyek ini adalah website portfolio sederhana yang terdiri dari beberapa halaman:
- Beranda
- Portfolio
- Prestasi
- Contact

## Cara Menjalankan
1. Buka `index.html` di browser.
2. Navigasikan halaman menggunakan navbar.

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
